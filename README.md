# Quote Generator 🌟

A CLI tool that displays random motivational quotes. Perfect for your daily inspiration!

## Features
- 🎨 Color-coded output
- 📥 Auto-creates quote database
- ➕ Add custom quotes
- ⏲️ Cron job ready

## Installation
```bash
git clone https://github.com/yourusername/quote-generator.git
cd quote-generator
chmod +x setup.sh
./setup.sh
